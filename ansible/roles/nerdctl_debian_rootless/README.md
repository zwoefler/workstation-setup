# Ansible Role: Nerdctl rootless install

Install rootless Nerdctl on Debian/Ubuntu systems.

Tested on:
- Ubuntu 24.04 LTS

### Requirements
None.

### Dependencies
None.

### Role Variable
None.

### Example Playbook
```YAML
- hosts: all
  roles:
    - role: nerdctl_debian_install
```
